# Clone dari repository
```
git clone URL
```

# Melihat status repository
```
git status
```

# Menambahkan file ke repository
```
git add nama_file
```
```
git commit -m "komentar"
```

# Membuat cabang baru 
```
git branch NAMA_BRANCH
```

# Cek daftar cabang baru atau branch 
```
git branch
```

# Pindah cabang
```
git checkout NAMA_BRANCH
```

# Menggabungkan cabang 
```
git merge nama_file
```
